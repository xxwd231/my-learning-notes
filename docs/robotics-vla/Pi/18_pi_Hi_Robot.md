## 一、 核心理念：系统 1 与 系统 2 的分层协同

Hi Robot 借鉴了人类大脑的“双系统认知理论”，将复杂的具身智能任务分为两级推理流程：

-   **系统 2（高层语义推理 - VLM）**：
    
    -   类似于人类慢速、深思熟虑的逻辑思考。
        
    -   负责解析开放式/复杂的长程指令（如“帮我清理桌面上的垃圾”），通过“自我对话”进行任务拆解与逻辑推理。
        
-   **系统 1（低层本能控制 - VLA / $\pi_0$）**：
    
    -   类似于人类快速、本能的肌肉记忆与动作反应。
        
    -   接收高层拆解出的原子子目标，以高频生成连续控制信号驱动机械臂运动[cite: 9]。
        

## 二、 高层策略 (VLM) 与 低层策略 (VLA) 对比

### 1. 高层策略 (VLM)[cite: 9]

-   **输入**：摄像头图像 + 用户长程指令 / 实时插话打断[cite: 9]。
    
-   **输出**：拆解后的低层语言命令（子目标指令） + 语音回复[cite: 9]。
    
-   **基座**：PaliGemma-3B，进行 VLM 专门微调[cite: 9]。
    
-   **运行频率**：**每秒一次** / 新交互触发（低频长时思考）[cite: 9]。
    

### 2. 低层策略 (VLA)[cite: 9]

-   **输入**：摄像头图像 + 低层语言命令 + 机器人关节状态[cite: 9]。
    
-   **输出**：连续动作块（Action Chunk，用于关节轨迹控制）[cite: 9]。
    
-   **基座**：$\pi_0$ (Flow Matching 连续动作专家)[cite: 9]。
    
-   **运行频率**：**实时控制**（高频毫秒级控制，如 50Hz）[cite: 9]。
    

## 三、 解耦架构优势与系统价值

-   **自然语言解耦（Language Interface）**[cite: 9]：
    
    -   两层之间**仅通过自然语言命令**进行衔接[cite: 9]。
        
    -   高层策略将大目标下发为子指令（如“先抓取左边的切片面包”），低层策略只需专注把当前的单步动作做精准。
        
-   **独立训练与模块化升级**[cite: 9]：
    
    -   **高层 VLM** 可借助大语言模型的强推理能力升级迭代，学习复杂的场景逻辑与人机打断逻辑[cite: 9]；
        
    -   **低层 VLA** 可专注在物理世界做运动学/动力学泛化微调，两层互不干扰[cite: 9]。
        
-   **灵活应对动态干预**：
    
    -   当用户突然插话（如“等等，把那个放到左边而不是右边”）时，高层策略可实时捕获并重新生成 low-level 指令，直接打断并指导低层策略修正动作[cite: 9]。
